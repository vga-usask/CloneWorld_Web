Preparation:
 * Have .NET core 2.1 installed.
 * Have R installed.
 * Make a directory named as "in".
 * The structure of "in" directory should be:
	- in
		- data_jsons
			- main.json
			- support_count_map.json
		- source_code
		- vis_system
* Get the path of Rscript.

Execution:
 * Type "dotnet run {1}" in the command line.
 * {1} should be replaced by Rscript path.
